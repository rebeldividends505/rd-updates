**APIs to hit (pick one, fall back to another):**

| Asset | Symbol | Primary API | Fallback API |
|---|---|---|---|
| Silver | SLV | Yahoo Finance (yfinance) | Alpha Vantage |
| Gold | GLD | Yahoo Finance (yfinance) | Alpha Vantage |
| Energy | XLE | Yahoo Finance (yfinance) | Alpha Vantage |
| NVDA | NVDA | Yahoo Finance (yfinance) | Polygon.io |
| Bitcoin | BTC-USD | CoinGecko Pro | CoinMarketCap |
| S&P 500 | SPY | Yahoo Finance (yfinance) | Alpha Vantage |
| QQQ | QQQ | Yahoo Finance (yfinance) | Alpha Vantage |

**Python pull example (yfinance, free):**

```python
import yfinance as yf
import pandas as pd

symbols = ['SLV', 'GLD', 'XLE', 'NVDA', 'SPY', 'QQQ', 'BTC-USD']
ytd_returns = {}

for sym in symbols:
    ticker = yf.Ticker(sym)
    # Get from Jan 1 2026 close to most recent close
    hist = ticker.history(start='2025-12-31', end='2026-05-19')
    if len(hist) > 0:
        start_close = hist['Close'].iloc[0]
        latest_close = hist['Close'].iloc[-1]
        ytd_pct = (latest_close / start_close - 1) * 100
        ytd_returns[sym] = round(ytd_pct, 1)

# Output for HTML patching
print(ytd_returns)
# Example: {'SLV': 22.5, 'GLD': 17.1, 'XLE': 13.3, 'NVDA': 16.8, ...}
```

**Output:** `data/ytd-snapshot-may19.json`
